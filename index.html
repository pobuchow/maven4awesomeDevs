default

  • validate — walidacja, czy projekt sie kompiluje
  • compile — kompilacja projektu
  • test — wykonanie testow automatycznych
  • package — stworzenie plików JAR lub WAR
  • integration-test — wykonanie testów integracyjnych
  • verify — weryfikacja metryk
  • install — umieszczenie artefaktu w repozytorium lokalnym Mavena
  • deploy — umieszczenie artefaktu na serwerze

konwencja nazw

  • target: katalog roboczy
  • src: pliki źródłowe
  • src/main: źródła, które trafiają do głównego artefaktu
  • src/test: źródła testowe
  • src/main/java: pliki źródłowe java
  • src/main/webapp: pliki źródłowe Web
  • src/main/resources: niekompilowalne pliki źródłowe
  • src/test/java: pliki testowe java
  • src/test/resources: niekompilowalne pliki testowe
